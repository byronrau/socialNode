Facebook API

This form uses Facebook API. Enter the URL of the page and it will return each post and all comments for that post. Only retrieves last 500 posts per page. If need to retrieve posts earlier enter the date in until field in format yyyy-mm-dd (ex 2014-12-31). This will get only posts until that date so it can retrieve up to 1000 posts previous. Post before 2011 may not include comments. For more detail see here.
